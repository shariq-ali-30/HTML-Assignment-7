# Chess Board

## About The Project

This project is a basic chess board created using HTML.

The purpose of this assignment was to practice HTML tables, rows, columns, images, and using attributes to create a structured chess board layout.

## Technologies Used

- HTML5

## Features

- 8x8 chess board layout
- Black and white chess pieces
- Alternating chess board colors
- Images used for chess pieces
- Table-based layout design

## What I Learned

Through this assignment, I learned:

- How to create layouts using HTML tables
- How to use rows and columns
- How to add images in HTML
- How to use attributes like:
  - bgcolor
  - width
  - height
- How to organize visual elements using HTML
